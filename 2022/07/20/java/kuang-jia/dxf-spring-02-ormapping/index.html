ORM

JPA、Hibenate、Mybatis

JPA是一种ORM思想的规范，类似于接口。而Hibenate是基于JPA的实现，类似于实现类。SpringData JPA是在Hibenate的基础上，对其更上层的封装实现。

Mybatis是一种半自动化的ORM框架，相较于JPA而言更灵活

Mongo

1.简要介绍一下MongoDB在Docker中的安装

1. 拉取镜像
docker pull mongo

2. 启动镜像
docker run --name mongo -p 27017:27017 -v ~/docker-data/mongo:/data/db -e MONGO_INITDB_ROOT_USERNAME=root -e MONGO_INITDB_ROOT_PASSWORD=root -d mongo

3.进入镜像
docker ex -it mongo bash

2.mongo常用命令

1. 登陆mongo
mongo -u root -p root

2. 查看库
show dbs

3.创建库
use springbucks；

4.创建用户
db.createUser (
    {
        user: "root",
        pwd: "root",
        roles: [
            {role: "readWrite", db: "springbucks"}
        ]
    }
)

5. 查询用户
show users;
  1. 测试当前用户是否能够登陆某个数据库
1. 切换库
use springbucks;

2. 使用某个用户登录
db.auth("username", "password");
  1. 查询某个文档
db.documentName.find();

Redis

操作Redis的手段

  • jedis
    • 非线程安全
    • 一般使用jedis pool
  • spring data redis

操作Redis步骤

  1. 导入依赖

    <dependency>
    <groupId>redis.clients</groupId>
    <artifactId>jedis</artifactId>
    <version>4.2.3</version>
</dependency>

  2. 增加配置

    redis.host=192.168.56.10
redis.maxTotal=5
redis.maxIdle=5
redis.testOnBorrow=true

  3. 获取Jedis Pool

    @Bean
@ConditionalOnMissingBean
public JedisPoolConfig jedisPoolConfig() {
    return new JedisPoolConfig();
}

@Bean(destroyMethod = "close")
public JedisPool jedisPool(@Value("${redis.host}") String host) {
    return new JedisPool(jedisPoolConfig, host);
}

  4. 获取Jedis连接客户端

    Jedis jedis = jedisPool.getResource()
